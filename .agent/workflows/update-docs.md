# 更新相关文档

当项目有重大更改时，使用此工作流更新所有相关文档。

## 需要更新的文档

1. **`.agent/OVERVIEW.md`** - AI Agent 上下文概览
   - 更新日期
   - 核心技术栈
   - 重要路径
   - 当前状态和已知问题
   - 编辑器功能

2. **`README.md`** - 项目主文档
   - 功能特性列表
   - 当前进度 checkbox
   - 项目结构说明

3. **`apps/backend/README.md`** - 后端服务文档
   - API 接口列表
   - 配置说明

4. **`SyncDesign.md`** - 同步架构设计
   - 数据库 Schema
   - API 协议
   - 内容格式说明

## 执行步骤

1. 读取上述 4 个文档
2. 分析当前代码变更（特别是新增功能、API、组件）
3. 更新文档内容，确保：
   - 日期正确
   - 功能列表完整
   - 技术栈准确
   - 已完成的任务标记为 `[x]`
4. 保存更新后的文档

## 注意事项

- 保持中英文一致性（README 主要中文，OVERVIEW 主要英文）
- 更新 `Last Updated` 日期
- 新增功能需要同时更新多个文档
- 保持 Markdown 格式整洁
