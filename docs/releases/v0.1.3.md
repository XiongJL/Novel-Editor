# v0.1.3

本次版本聚焦 GitHub Actions 发布链路修复。

## 亮点

- 修复 `actions/setup-node` 在 pnpm 尚未安装时就尝试初始化 pnpm 缓存，导致 Windows 与 macOS job 在 `Setup Node.js` 阶段失败的问题。
- 保留 runner 上直接安装 `pnpm@8.15.9` 的方案，减少第三方 action 拉取失败带来的不确定性。
- 发布流程继续由 tag push 自动触发并自动创建 GitHub Release。

## 下载

- Windows 安装版：`云梦小说编辑器-Setup-0.1.3.exe`
- Windows 便携版：`云梦小说编辑器-Portable-0.1.3.exe`
- macOS DMG：`云梦小说编辑器-mac-0.1.3.dmg`

## 说明

- 触发发布请推送版本标签，例如 `git push origin v0.1.3`。
- 工作流成功完成后，Release 页面会自动附加 Windows 与 macOS 产物。
