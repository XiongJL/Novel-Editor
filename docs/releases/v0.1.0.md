# v0.1.0

首个公开发布版本，当前聚焦桌面端离线创作体验、结构化创作工具与 AI 辅助写作闭环。

## 亮点

- 支持 Windows 安装版、Windows 便携版与 macOS DMG 发布。
- 集成 Lexical 富文本编辑器，覆盖卷章管理、排版、快捷键与自动保存。
- 支持灵感系统、故事线/情节点/锚点、叙事矩阵与全局搜索。
- AI 创作工坊支持生成草稿、编辑勾选、入库前校验与原子事务写入。
- 章节续写支持配置参数、预览确认后插入，摘要支持本地与 AI 双模式。

## 下载

- Windows 安装版：`云梦小说编辑器-Setup-0.1.0.exe`
- Windows 便携版：`云梦小说编辑器-Portable-0.1.0.exe`
- macOS DMG：`云梦小说编辑器-mac-0.1.0.dmg`

## 说明

- AI 的 `API Key` / `API Token` 仅保存在当前设备本地，不写入小说数据库，也不随备份恢复迁移。
- macOS 安装包如未签名或未公证，首次打开可能会遇到系统安全提示。
- 当前版本仍在持续完善标题生成进度反馈与云同步完整对接。
