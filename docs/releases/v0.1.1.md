# v0.1.1

本次版本聚焦发布链路稳定性与 Windows 图标一致性修复。

## 亮点

- 修复 Windows 资源管理器中的主程序 `.exe` 仍显示默认 Electron 图标的问题。
- `build:win` 现在会先修正 `win-unpacked` 主程序图标，再基于该目录生成安装版和便携版。
- 打包结果中的窗口图标、任务栏图标、快捷方式图标与最终 `.exe` 图标保持一致。
- 仓库与 GitHub Actions 的 pnpm 基线统一到 `8.15.9`，并修正文档中的初始化命令。

## 下载

- Windows 安装版：`云梦小说编辑器-Setup-0.1.1.exe`
- Windows 便携版：`云梦小说编辑器-Portable-0.1.1.exe`
- macOS DMG：`云梦小说编辑器-mac-0.1.1.dmg`

## 说明

- 初始化仓库请使用 `pnpm install` 后接 `pnpm run setup`。
- Windows 打包命令为 `pnpm --filter novel-editor-desktop run build:win`。
- 若 macOS 构建为未签名产物，首次打开仍可能遇到系统安全提示。
