# v0.1.2

本次版本聚焦发布流程稳定性修复。

## 亮点

- 修复 GitHub Actions 在 macOS job 初始化阶段下载 `pnpm/action-setup` 失败的问题。
- Release 工作流改为在 runner 上直接安装 `pnpm@8.15.9`，减少第三方 action 拉取失败带来的不确定性。
- 保持 Windows 与 macOS 发布流程一致，继续由 tag push 自动创建 GitHub Release。

## 下载

- Windows 安装版：`云梦小说编辑器-Setup-0.1.2.exe`
- Windows 便携版：`云梦小说编辑器-Portable-0.1.2.exe`
- macOS DMG：`云梦小说编辑器-mac-0.1.2.dmg`

## 说明

- 发布命令仍然是推送版本标签，例如 `git push origin v0.1.2`。
- 若工作流成功完成，GitHub Release 会自动附加 Windows 与 macOS 产物。
