# 小说编辑器用户指南

## 1. 界面概览
左侧活动栏用于切换主要功能：
- 资源管理器：卷章管理
- 灵感：灵感记录与检索
- 故事线：情节线与情节点
- 世界观：角色/物品/设定/地图
- 搜索：全局搜索
- AI 创作工坊：AI 生成草稿并确认入库
- 设置：主题、语言与 AI 配置

## 2. 基础写作操作
- 新建卷/章：在资源管理器点击 `+`
- 重命名：双击卷名或章名
- 打开章节：单击章节
- 自动保存：编辑内容会自动保存
- 常用排版：工具栏支持加粗、对齐、字号、排版等

## 3. AI 功能

### 3.1 标题生成
1. 在章节标题区域点击“AI 生成标题”。
2. 系统会基于当前章节内容给出候选标题。
3. 选择候选即可替换标题。

说明：标题生成进度条正在开发中，当前版本先提供基础 loading 态。

### 3.2 章节续写（推荐流程）
1. 点击“续写”按钮打开配置弹窗。
2. 设置目标字数、创意程度、参考章节范围、风格语气。
3. 可选关联灵感（多选）。
4. 生成后先查看预览，再确认插入正文。

首章规则：若第一章没有大纲，系统会引导你先建大纲或先手写开头。

### 3.3 AI 创作工坊
1. 打开活动栏“AI 创作工坊”。
2. 输入创作需求（题材、风格、核心冲突等）。
3. AI 生成草稿（大纲、角色、物品、技能、地图）。
4. 在工坊内编辑、删除、勾选要入库的条目。
5. 点击确认入库。

说明：入库前会做校验；若某项失败，本次会按事务整批回滚，避免部分写入。

### 3.4 摘要策略
- 本地摘要：不消耗模型配额，速度快。
- AI 摘要：质量更高但会消耗模型配额。
- 推荐：AI 摘要使用“手动生成”或“章节完稿后触发”。

### 3.5 AI 密钥保存说明
- `API Key` / `API Token` 仅保存在当前设备本地。
- 保存位置为应用用户目录下的 `ai-settings.json`。
- 软件不会把这些密钥写入小说数据库。
- 备份/恢复功能不包含这些密钥，恢复备份后如需继续使用 AI，需要重新填写。

## 4. 常用快捷键
| 功能 | Windows/Linux | macOS |
| --- | --- | --- |
| 保存 | `Ctrl + S` | `Cmd + S` |
| 撤销 | `Ctrl + Z` | `Cmd + Z` |
| 重做 | `Ctrl + Y` | `Cmd + Shift + Z` |
| 页内查找 | `Ctrl + F` | `Cmd + F` |
| 全局搜索 | `Ctrl + Shift + F` | `Cmd + Shift + F` |
| 新建灵感 | `Ctrl + I` | `Cmd + I` |

## 5. 常见问题
- 续写结果与预期不一致：先调整“参考章节范围”和“风格语气”，再生成。
- 搜索不到内容：确认章节已保存后重试。
- 地图生成失败：检查模型配置与网络后再重试。
- 更换设备或恢复备份后 AI 配置丢失：这是预期行为，`API Key` 仅本地保存，不随备份迁移。
